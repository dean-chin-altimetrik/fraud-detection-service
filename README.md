# Fraud Detection Service

A microservice for detecting and preventing fraudulent transactions.

## Features

- Real-time fraud detection
- Transaction risk scoring
- Pattern recognition
- Alert generation
- Fraud case management

## API Endpoints

- `POST /fraud/check` - Check transaction for fraud
- `GET /fraud/alerts` - Get fraud alerts
- `POST /fraud/report` - Report suspected fraud
- `GET /fraud/risk-score/{transaction_id}` - Get risk score

## Technology Stack

- Python 3.9+
- FastAPI
- Machine Learning models
- Redis
- PostgreSQL
